## 🚀 Comment Démarrer

### Prérequis
- Node.js (v14+)
- MongoDB
- npm ou yarn

### Installation

1. Clonez ce dépôt
   ```bash
   git clone https://github.com/votre-username/devsiteapp.git
   cd devsiteapp
   ```

2. Installez les dépendances
   ```bash
   npm install
   ```

3. Configurez les variables d'environnement
   ```bash
   cp .env.example .env
   # Modifiez le fichier .env avec vos configurations
   ```

4. Démarrez le serveur de développement
   ```bash
   npm run dev
   ```

5. Accédez à l'application via `http://localhost:3000`
